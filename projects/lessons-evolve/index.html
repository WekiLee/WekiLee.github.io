Overview

概述

将工程实践中反复遇到的坑和教训系统化管理，通过 lesson → pattern → instinct 三级进化机制，把一次性的故障修复逐步沉淀为可自动执行的预防规则，形成团队的长期经验资产。

Architecture

架构设计

采用分层进化架构，从记录到验证再到自动化执行，让经验从文档变成代码。

采集层

故障复盘记录代码审查发现线上事故总结

进化层

Lesson 分类Pattern 提取Instinct 转化

执行层

自动检查规则CI 集成PR 阻断

Features

核心特性

🗺️

三级进化

lesson（临时记录）→ pattern（验证后的模式）→ instinct（自动执行的规则），层层递进。

🔄

不只是文档

最终落地为可执行的 CI 检查规则，在代码层面自动阻断同类错误。
