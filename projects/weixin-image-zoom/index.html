Overview

概述

一个轻量级 JavaScript 脚本，专门解决微信公众号文章内嵌图片过小的问题。安装后点击文章中的任意图片即可放大查看，体验类似手机相册。零依赖注入，不拖慢页面加载。

Architecture

架构设计

纯 JavaScript 实现，通过监听 DOM 中的图片点击事件，动态创建全屏遮罩层实现图片放大效果。

注入层

TampermonkeyContent Script

核心逻辑

事件委托DOM 操作CSS 动画

兼容层

Features

核心特性

🗺️

零依赖

不依赖 jQuery 或任何第三方库，纯原生 JS 实现，体积不到 3KB。

🔄

一次安装，到处生效

通过 Tampermonkey 安装后，所有公众号文章自动启用，无需额外操作。
