ImageTransformer

AI 智能证件照生成工具

稳定版
GitHub

概述

ImageTransformer 是一个 Windows 桌面工具，利用 ONNX 深度学习模型实现精准人像分割。无需联网，本地即可一键完成证件照的换底、裁剪、压缩和排版打印。

架构设计

基于 .NET 8 + WPF 构建桌面应用，集成本地 ONNX 推理引擎，无需 GPU 即可运行 AI 抠图。

UI 层

  • WPF MVVM
  • Material Design
  • 实时预览

AI 引擎

  • ONNX Runtime
  • RMBG-1.4
  • u2netp

图像处理

  • ImageSharp
  • SkiaSharp
  • 色彩空间转换

输出层

  • 多规格排版
  • DPI 自适应
  • JPEG/PNG 导出

核心特性

双引擎 AI 抠图

内置 RMBG-1.4 和 u2netp 两种模型，可根据图片类型切换，准确率与速度兼顾。

无网可用

所有 AI 推理均在本地完成，无需上传图片到云端，保护隐私且离线可用。

排版打印

内置 5寸/6寸/A4 三种排版方案，直接导出即可打印，省去照相馆排队的麻烦。

  • .NET 8
  • ONNX Runtime

前端
