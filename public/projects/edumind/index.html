EduMind

开源 AI 驱动个人导师

活跃开发中
GitHub

概述

EduMind 是一个开源的 AI 驱动个人学习系统。核心理念是：学习应该有一条清晰的路径。它通过知识图谱将零散的知识点组织成可导航的拓扑结构，再结合 AI 导师的个性化引导，让每个人都能从零到一建立起完整的知识体系。

架构设计

EduMind 采用前后端分离架构，后端以 FastAPI 为核心，Neo4j 存储知识图谱，前端基于 React 18 构建交互式学习界面。

前端层

  • React 18
  • TypeScript
  • 知识图谱可视化

API 层

  • FastAPI
  • WebSocket 实时通信
  • RESTful 接口

数据层

  • Neo4j 知识图谱
  • PostgreSQL
  • Redis 缓存

AI 层

  • LLM 多模型切换
  • LlamaIndex
  • 领域感知策略

核心特性

知识图谱驱动

每个知识节点相互关联，自动生成拓扑序学习路径，告别碎片化学习。

教、练、评、拓一体化

从概念讲解到随堂练习，再到综合评测和能力拓展，形成完整学习闭环。

AI 模型自由切换

Web UI 界面即可切换底层 AI 模型，不被任何单一厂商锁定。
